Commit Using Git Shell
  1. Open Up Git Shell
  2. Open up the Directory your Github File is open
  3. Type "Git Status" to see if you have any files that have not been committed or synced
  4. If you do have files that needs to be committed, then type "git commit -m -a 'This is what my commit will be called.'"
  5. Push your commits to Github by typing "git push"
Create a Second site on your Github Repository
  1. Navigate to Github.com and log into your website
  2. Go on Github and see if you have just a master branch.
  3. If you have just a master branch, create a new branch
